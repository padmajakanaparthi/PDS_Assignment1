The dataset examines the frailty of female individuals and includes the following fields:

Height (in inches): Represents the height of each individual.
Weight (in pounds): Indicates the weight of each individual.
Age (in years): Denotes the age of each individual.
Grip Strength (in kilograms): Reflects the grip strength of each individual measured in kilograms.
Frailty (Y/N): This field indicates whether an individual is considered frail or not, with values 'Y' for yes and 'N' for no.
